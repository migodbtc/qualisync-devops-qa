# Dashboard Features

---

## Customized Role Dashboard

- The dashboard adapts dynamically based on user roles:
    - **Admin:** Full system control and oversight.
    - **Finance:** Finance-focused analytics and reporting.
    - **Staff:** Operational management tools.
    - **Tenant:** Personalized dashboard with access to personal data and requests.

---

## Card-Based Tenant Directory

- Main layout uses a 70/30 split:
    - **70%:** Directory index.
    - **30%:** Sidebar for quick details and additional operations.
- Sidebar includes a "Show All Info" modal for expanded tenant details.
- Features:
    - Register new tenant.
    - Export directory to CSV.
    - Search bar, pagination, and advanced filtering.

---

## Apartment Rooms

- View all rooms with options to edit details.

---

## Payment History

- Timeline view of all payments.
- Features:
    - Search bar, pagination, and filtering.
    - Modal to magnify payment details, including user, room, and transaction specifics.

---

## Finance Analytics

- Separate from payment history (which serves as an audit log).
- Includes:
    - Statistics on overdue payments.
    - Revenue amounts (by month, year, week, etc.).
    - Revenue sources (by payment type).
    - List of overdue users or users nearing payment deadlines, with options to email or chat.

---

## Email Communications

- Email individual or multiple users.
- Use pre-generated templates for common communications.
- Standard email formatting tools.
- Automated email footers.

---

## User Profile Management

- Edit personal information.
- View full account information.
